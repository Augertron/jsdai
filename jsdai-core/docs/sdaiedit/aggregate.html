Aggregate page


This page shows one aggregate, which can be attribute value or sub value ofother aggregate. At the top type of aggregate are given in EXPRESS style. Types can be BAG, SET, LIST and ARRAY. Operations and view are a little bit different for all these types. After that all members are given. ForARRAY types keywords 'UNSET' are given for non set members. At the bottomare line where you can enter new value. By pressing 'Add' button value fromthis field will be added at the selected position. 'Get' and 'Set' buttonsare for sdai clipboard operations. With get you can get selected element and with set  add add new element. 'Remove' button will remove or unset selectedelement.
