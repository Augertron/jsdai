JSDAITM is a toolkit and library for the development of STEP applications according to the international standard ISO 10303. It can also be used for the implementation of any other data model based on the EXPRESS modelling language (ISO 10303-11) such as PLIB (ISO 13584) and, IEC61360-Standard_data_element_types

The intendended audiance of this package are developers, working on software, datamodels and example data sets. Readers should be familiar with STEP data models, the underlying EXPRESS modelling language, and the Java programming languange. It is not indented for end-users.
