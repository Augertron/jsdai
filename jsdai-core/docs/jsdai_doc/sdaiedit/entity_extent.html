Entity Extent Page



The EntityExtent page represents data related to one entity data type (entity definition). Data can be merged from several models (in the case that you are working on the schema instance instead of the model). The list of models is situated in the Model combo box on the top section of the window.
The entity data type name (entity definition name) is represented in the Definition field. The current model in the Model combo box will be changed automatically to the respective instance model after you select an instance. All instances are represented in the Instances list. They show types available in the model. If you want to create an instance of this type, you need to select model in which you will create it and press 'Create' button. If you want to delete an instance you need to select it and press 'Delete' button. 'Get' button is for further editing operations. If you select an instance and press 'Get' then selected instance is placed in the SDAI clipboard and you are be able to use 'Add' operation later. More for editing purposes see entity instance.
