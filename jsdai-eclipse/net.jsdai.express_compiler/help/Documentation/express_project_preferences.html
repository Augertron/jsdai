Express Project Preferences


You can change some global settings for Express project by going to Window -> Preferences -> JSDAI -> Express project dialog window:
  • Create template .exl file for express file list when creating new express projects;
  • Create template ..._excluded.exl file for exclusion list when creating new express projects;
  • location of temporary files for Express projects:
    • in system temp - use system default temp location, deleted when Eclipse is closed;
    • temp in Eclipse workspace - use project-life-long-persistent temp location in the Eclipse workspace;
    • temp directory in Express project - use Temp subdirectory inside the express project.


Figure 1. Preferences dialog window, Express projects panel
Note.    You can set your own  settings for an Express project while creating a new  project or by going to already created Express project preferences (right click on the project and select Properties -> Express -> Temporary files). Exception: You can not select to create or not exl files for the already created Express project, because they could be created only at once - while creating the project.
