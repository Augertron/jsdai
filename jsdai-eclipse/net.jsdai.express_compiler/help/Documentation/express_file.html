Express file (.exp) contains one or more EXPRESS schemas. It is recommended to have one schema in one express file and file should be named according to the schema.
To open Express file in the Express editor double click on the file with the extension .exp in the Navigator panel.

Launching  New Express File wizard

After an Express project is created, you can create a new Express file. First of all you mus click the mouse button on the Express project folder area.
You can launch the New Express File wizard (see Figure 1) in the following ways:
  • If the Express perspective is switched on:
    •  use the (Create new Express File with .exp extension) button on the Eclipse tool bar.
Or
    • choose from the main menu bar: Express -> New Express File.
                   Or
    • Select File -> New ->  Express File.
  • Select File -> New -> Other -> JSDAI ->  Express File.

Creating Express File

After the New Express File wizard is launched, the particular dialog window appears (see Figure 1).
Follow the routine to create a new Express file:
  • Enter the directory of the Express file to be located into the Container field or select one by using the Browse button. Note: The default directory of the currently selected project is provided automatically.
  • Enter a name of the file to be created (with an extension .exp) into the File name field..
  • Click the Finish button to create an Express file. The Express editor panel is activated automatically, containing the created file.
