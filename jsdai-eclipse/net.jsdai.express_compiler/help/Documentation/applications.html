Using JSDAI libraries in Java applications

When the JAR file is created with the library containing JSDAI java packages for the Express schema(s) you compiled, you may use that JAR file in your java applications.
For that purpose, you may create an JSDAI Java project, write a JSDAI application, import into it the java packages you need (contained in the JAR file) and use them in your application.
For that you need the following things:

  1. Open Java Project Properties dialog, Java Build Path window, Libraries panel:
    • Right click on the JSDAI Java project in the Navigator panel. The context sensitive menu appears.
    • Select the Properties menu item. The Properties for Java project dialog window appears (see Figure 1).
    • Select the Java Build Path node in the tree on the left and click the Libraries tab on the right side of the window.


    Figure 1. Properties for Java project dialog window, Java Build Path panel view

  2. Add JAR file as a library to Java Build Path in the properties of the java project:

      • Click the Add JARs... button to add your JSDAI library. The JAR Selection dialog window appears (see Figure 2).
      • Select an applicable JAR file in the presented hierarchy (it will be available to be added if you have it in the Express project in the same workspace) and click the OK button. The selected JAR appears in the Libraries panel.


    Figure 2. JAR Selection dialog window view
