Input .exl file list

This EXL file can be created manually, or while creating a new Express project, if particular option is set in Express project references. It contains the list of names with full paths of express files to be included into the compilation.
The names of files you may add in the following ways:
  • Drag and drop express files to the Editor panel from the Navigator panel;
  • Add files with full paths manually.
Also you may comment out the files that are to be temporarily excluded from the compilation.
