Select object

You can select (highlight) object directly on the diagram in the Editor or in the Express-G Outline window:
  • Single mouse click: Select the object.
  • <Shift> key + Single mouse click: The <Shift> key is held down while making a selection, all of the newly selected objects have their selection state reversed.
  • Drag with mouse: Make a rectangle, objects that are inside the rectangle will be selected. Also place an object onto a diagram.
  • <Ctrl> key + A : Select all objects presenting in the current diagram page (Editor window).
