Fonts

You can set an applicable fonts for the schema objects in the Editor panel. The fonts can be changed for the current schema or for the schemas to be created (new schemas default).
  • Select the Window --> Preferences drop down menu item. The Preferences window appears (see Figure 1).
  • Select the JSDAI --> Editors --> Express-G --> Diagram  on the left side of the window to change the fonts of  the schemas to be created in the future:
          Or
  • Select the JSDAI --> Editors --> Express-G --> Diagram --> Current diagram - to change the fonts of current schema objects.             
  • A new panel (Diagram (see Figure 1) or Current diagram(see Figure 2)) appears on the right side of the window.
  • Click the Change button next to the Entity name font field to change an entity fonts or next to the Attribute name font - to change attributes fonts. Note: The current fonts are presented next to the particular field.
  • The Font window appears to select an applicable fonts (see Figure 3). After the fonts are selected, click the OK button and exit the window.
  • Click the Apply button on the bottom section of the panel to apply changes. If you want to restore default ones, click the Restore Defaults button.
  • After all selections are made exit the Preferences window by clicking the OK button.
