Arrange

Adjust the spaces between selected objects:
  • Horizontally
  • Vertically

Arrange Horizontally

Follow the routine to evenly arrange selected objects (see Figure 1) horizontally between the first and the last selected object, left to right:
  • Right click on the selection area. The context sensitive menu appears.
  • Select the Arrange --> Horizontally menu item. Selected objects are arranged horizontally (see Figure 2).

Arrange Vertically

Follow the routine to arrange selected objects (see Figure 1) vertically between the first and the last selected object, top to bottom:
  • Right click on the selection area. The context sensitive menu appears.
  • Select the Arrange --> Vertically menu item. Selected objects are arranged vertically (see Figure 3).
