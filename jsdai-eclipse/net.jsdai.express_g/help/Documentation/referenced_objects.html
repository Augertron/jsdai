Referenced Objects

  • Bring to this page
  • Remove from this page

After you add any schema object from the Express-G Outline panel onto the Editor panel the interfaced schemas are represented in the rounded boxes (see Figure 1). You can bring those schemas onto the current diagram or remove them if they are already there.

Bring to This Page

Follow the routine to bring the referenced object onto the diagram:
  • Select the composition (page reference) symbol on the diagram (see Figure 1).

Figure 1. Selected composition symbol.
  • You can either:
    • Right click on the selected symbol in the Editor panel. The context sensitive menu appears.
    • Select the Bring to this page --> Users --> all / <listed objects>. Note: If you select all menu item - all users of the selected entity are placed on the current page. You can place only one object on the current page by selecting it from the list in the context sensitive menu.
    • All users or the selected one with all attributes are presented on the current page (see Figure 2).
Or
    • Click CTRL key and double click on the referenced object.
Or
    • Select the referenced object in the Express-G Outline panel and place it on the page,



Figure 2. Entity ('Activity') with attributes is placed on the current page.

Remove from this Page

To remove a referenced object from the diagram simply remove it from the diagram.
