EXPRESS-G Outline Window

EXPRESS-G Outline window presents the objects of the selected schema in the Outline panel (see Figure 1): :
  • - interfaced schema, with   interfaced data types if any.
  • Main schema (selected in the Outline panel) with presenting objects:
    • Defined types (static node):
      • Defined type,
      • Select type,
      • Enumeration.
    • Entity types (static node):
      • Entity with relationships if any.
    • Constraint types (static node):
      • Constraint type.
Note: If (for complete schemas) or (for partial schemas) icon is on the object icon, it means the object is not placed on the diagram (Editor panel). Number in brackets (e.g. (1)), next to the object name indicates the page number the object is represented on the Editor panel.
