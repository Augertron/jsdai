Move Page

Follow the routine to move page content to another page: in the Editor panel
  • Right click on the page tab or on the page line in the info table list. The context sensitive menu appears (see Figure 1).
  • Select the Move menu item. The dialog window appears. Refer the Move to Page for details.
  • The page with it's content is moved to the entered one.
