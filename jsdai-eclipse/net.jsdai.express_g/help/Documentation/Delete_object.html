Delete Object

You can delete an object from the diagram by using Del key on the keyboard or Context sensitive menu:
  • Select an applicable object you want to remove in the Editor or Express-G Outline panel. The object is highlighted on the diagram (see Figure 1).
  • Press down the Del button.
        Or
  • Right click on the selected object. The context sensitive menu appears.
  • Select the Delete menu item.
The object will be deleted from the diagram.
