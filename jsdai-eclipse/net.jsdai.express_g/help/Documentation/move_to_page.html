Move to Page

Follow the routine to move the selected object to an applicable page of the diagram:
  • Right click on an applicable object on the diagram in the Editor panel or object name in the Express-G Outline panel. The context sensitive menu appears.
  • Select the Move to page menu item. The dialog window appears to select the page number (see Figure 1).
    • Select an applicable number of the page in the table.
    • Click OK button to confirm your choice or Cancel button to reject the selection and close the window.
