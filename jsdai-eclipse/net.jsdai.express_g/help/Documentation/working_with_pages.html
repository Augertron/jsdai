Working with pages

EXPRESS diagram can be formed from a number of pages. Each page has a number and name. The user has a possibility:
  • insert pages - to add a new page,
  • delete page  - to delete an incorrect or redundant page with all it's content,
  • rename page - to change a name of a page or create the missing one,
  • re-ordered - to move page into an applicable position,
  • resize - to adjust an applicable page size,
  • frame - to define editing area (margins), 
  • define page label  - to define an applicable text line on the bottom section of the page,
  • move object to another page - to move the content of the current page to the preferable page,
  • make page visible/invisible - to make a page visible or not for printing,
  • bring referenced objects onto page - to add interfaced schemas onto the page.
  • print pages - to print the pages of the current diagram.
