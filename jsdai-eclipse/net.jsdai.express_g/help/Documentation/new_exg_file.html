Create New File

There is a possibility to create a new .exg file in an applicable project. Follow the routine:
  • Select New --> Other menu item in the context sensitive menu or in the File drop-down menu. The New dialog window appears (see Figure 1).

Figure 1. New dialog window
  • Select the Express-G file (*.exg) node in the tree and click the Next button. The New Express-G file dialog window appears (see Figure 2).

Figure 2. New Express-G file dialog window
  • Enter or select the parent folder.
  • Enter the file name to be created into the File name field.
  • You must either:
    • Click the Next button to create a file, containing dictionary data. The next dialog window appears (see Figure 3).
    • Select the Import dictionary data at once option.
    • Enter or select the dictionary (.exd) file in the File field and click the Finish button.
Or
    • Click the Finish button to create an empty file.

Figure 3. Dialog window to select dictionary file
