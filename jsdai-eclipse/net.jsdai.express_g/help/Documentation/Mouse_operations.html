Mouse Operations

  • Left mouse button
  • Right mouse button

Left mouse button

Following actions can be performed by using the left mouse button:
  • Drag: Make a rectangle, objects that are inside the rectangle will be selected. Also Move an object onto a diagram.
  • Single click: Select the object.
  • <Shift> + Single click: The <Shift> key is held down while making a selection, all of the newly selected objects have their selection state reversed.
  • Double click: Open the object properties window.
  • <Ctrl> + Double click: Bring to the diagram page the interfaced objects. 

Right Mouse Button

This button always displays the context sensitive menu (see Context Sensitive Menu for more information).
