Create Schema Layout

Follow the routine to create a schema layout:
  • Right click on an applicable schema in the Outline panel.  The context sensitive menu appears (see Figure 1).

Figure 1. Context sensitive menu
  • Select one of them at a time:
    • New diagram --> Complete (Short/Long) menu item to create the Complete schema layout (short or long form):
      • Complete short layout
      • Complete long layout
    • New diagram --> Partial (Short/Long) menu item to create the Partial schema layout (short or long form).
      • Partial short layout
      • Partial long layout
The created schema layout node appears under the selected schema node in the Outline panel (see Figure 2). The content of the created schema layout is presented in the EXPRESS-G Outline panel (see Figure 3).


Figure 2. Complete (short and long) and Partial (short and long) under action_schema node


Figure 3. The content of the schema layout (complete short (left) and partial short (right))
