Outline

The Outline panel presents the hierarchy of schemas with diagrams (layouts) situated beneath (see Figure 1):
  • Current repository name with available schemas listed below:
    • schema name:
      • Complete short diagram,
      • Partial short diagram,
      • Complete long diagram,
      • Partial long diagram.
You can perform the following operation in the Outline panel:
  • Create new diagram (layout)
  • Delete diagram (layout)
