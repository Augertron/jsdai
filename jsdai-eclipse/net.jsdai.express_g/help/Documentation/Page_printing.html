Page Printing

You can print thepage  Editor  page panel content:
  • Activate the Editor window and click on the page you wish to print. The boundaries of the page should appear in red colour (see Figure 1).
  • Click the tool bar button or File --> Print menu. The Print dialog window appears. Select an applicable options, pages you want to print and click the Print button.

Figure 1. Activated page in Editor window

Note: The invisible pages will not be printed, you must make them visible first.
