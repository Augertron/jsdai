Context Sensitive Menu

A context sensitive menu offers only menu items that are applicable to the object or region at the location of the mouse pointer:
  • Properties - select this menu item to change object properties.
  • Make visible -  this menu is enabled after the invisible object is selected (simple data type, page).
  • Make invisible - this menu item is enabled after the simple data type object is selected.
  • show expressions - to show expressions of derived attributes instead of simple type square.
  • hide expressions - to hide expressions of derived attributes and show simple type square.
  • Reset placement hint - set an initial hint place of the relationship.
  • Auto size - reset an initial size of the object box.
  • Delete - remove the selected object from the diagram.
  • Move to page - move the selected object to the particular page.
  • Layout on this page - restore the initial layout of the select object on the diagram.
  • Justify - put objects into the particular order, according to top or left object.
  • Arrange -  adjust the spaces between selected objects
  • Bring to this page - add referenced users (all or selected from the list).
  • Grouping - bundle up/off similar objects.
