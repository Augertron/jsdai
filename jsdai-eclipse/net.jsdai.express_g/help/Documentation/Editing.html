Editing

This help menu provides information about the editing of objects in the Editor, Outline and Express-G Outline panels. A user has a possibility:
  • Create diagram - to create diagram for the selected schema in the Outline panel.
  • Delete diagram - to delete the selected diagram from the Outline panel. Note: all diagram content, represented in the Editor panel will be deleted.
  • Select object - to select/highlight an object on the diagram in the Editor and Express-G Outline panels.
  • Place object - to place an object from the Express-G Outline onto the Editor panel.
  • Delete object - to delete an object from the diagram in the Editor panel.
  • Change appearance of relationship line - to change initial shape and name location of an applicable relationship.
  • Make simple type invisible / visible - to change an appearance of the simple type object (visible or not).
  • Resize and autosize object box - to adjust an applicable object box size or autosize.
  • Change text fonts - to ajust new fonts for the object box label.
  • Arrange - to adjust spaces between objects.
  • Justify - to justify selected objects.
  • Group - to group similar objects (trees, interfaced objects, page references and simple types).
  • Properties - to view object properties.
