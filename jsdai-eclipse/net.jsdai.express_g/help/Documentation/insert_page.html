Insert Page

Follow the routine to add more pages or insert the missing one on the Editor panel:
  • Right click on the page tab or on the page line in the info panel table list. The context sensitive menu appears (see Figure 1).
  • Select the Insert menu item. The new page will be inserted before the selected page.
       Or
  • Click on '+' next to the page tabs to add a new page at the end of all pages.

Figure 1. Context sensitive menu

Note: The inserted page is of the same visibility, size, frame, text and label as the currently selected one.
