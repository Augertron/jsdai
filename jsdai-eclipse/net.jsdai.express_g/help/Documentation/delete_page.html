Delete Page

Follow the routine to remove a page from the Editor panel:
  • You can either:
    • Right click on the page tab or on the page line in the info tab table list. The context sensitive menu appears (see Figure 1).
    • Select the Delete menu item.
                Or 
    • Click on the page tab you want to delete.
    • Go to drop down menu on the top section of the workbench.
    • Select the File --> Delete menu item.
  • The selected page is removed from the diagram Editor panel.
