Page Size, Frame and Label

You can change the appearance of a page:
  • Size
  • Frame
  • Label
First of all you have to open Preferences dialog window (see Figure 1) by using the Window --> Preferences drop down menu item and select the JSDAI --> Editors --> Express-G  node on the left side of the window. The Express-G panel appears on the right side. After some  changes were made click the Apply or OK button to apply changes or the Restore Defaults button to set the default values.


Figure 1. Preferences window to change page size, frame and label

Size

You can change the width and height of a page. First of all open the Current Page frame panel (see Figure 2) by selecting JSDAI --> Editors --> Diagram --> Current Page frame node.


Figure 2. Preferences dialog window, Current Page frame panel
  • Lock same page size and visibility for all pages - select this option to set the same size and frame for all pages.
  • page width - enter the width of page in pixels,
  • page height - enter the height of page in pixels,
Also there is a possibility to change the page size directly on the Editor panel:
  • Click on the 'white' area of a diagram. The page boundaries are highlighted in red colour with squares on the corners (see Figure 3).
  • Point on the highlighted square and drag until the needed size of the page is reached.

Figure 3. Highlighted page boundaries

Frame

You can adjust an applicable frame for a page in the Current Page frame panel (see Figure 2):
  • top margin - enter the top margin position in pixels,
  • left margin - enter the left margin position in pixels,
  • right margin - enter the right margin position in pixels,
  • bottom margin - enter the bottom margin position in pixels.

Label

You can define an applicable label for a page - text on the bottom section of the page in the Current Page frame panel (see Figure 2):
  • Enter an applicable values into the field next to the Page frame text mask.
  • use default for current page - select this check box to set the default label for the current page. The default values can be changed in the below field. Note: You can enter a simple text (letters, symbols) and/or use the following parameters:
    • %name - page name,
    • %page - page number,
    • %max_page - total number of pages, forming the diagram.
For example: you wish to have the selected page  with a label 'Figure H.1 - activity_schema - EXPRESS-G diagram 1 of 2', where activity_schema - page name, 1-current page number, 2-total number of pages forming the diagram:
    • Select a page on the Editor window you want to apply a new label.
    • Select the Window --> Preferences drop down menu item, the JSDAI --> Editors --> Diagram --> Current Page frame  node on the left side of the window.
    • Enter the following line into the edit field: Figure H.%page - %name - EXPRESS-G diagram %page of %max_page. 
    • Set an option use default for current page option OFF (uncheck).
    • Click the Apply or OK button to apply changes. You can see a result of the example in Figure 4.
