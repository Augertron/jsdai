Dictionary data in p21 format


Follow the routine to export EXPRESS dictionary data to p21 format:
  • Select in the main menu bar File -> Export or right click in the Navigator view area and select Export menu item in the appeared context sensitive menu.
  • The Export wizard dialogue window appears on the screen.
  • Select Dictionary data in p21 format node under the JSDAI branch and press Next button. The Export EXPRESS Data to STEP file wizard dialogue window appears (see Figure 1):

Figure 1. Export to STEP wizard dialogue
    • Select Express Data file - locate a path to the EXPRESS dictionary file (.exd extension) by using Browse button.
    • Select STEP file - locate a path to the destination STEP file by using Browse button.
    • Press Finish button. The generated STEP file appears in the specified location.
