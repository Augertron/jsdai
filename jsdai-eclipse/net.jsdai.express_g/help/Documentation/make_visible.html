Make Simple type Visible / Invisible

  • Make invisible
  • Make visible

Make Invisible

Follow the routine to make the simple data type square invisible on the diagram:
  • Select the simple data type square on the diagram. The object is highlighted (see Figure 1)

Figure 1. Highlighted simple data type square
  • Right click on the selected object. The context sensitive menu appears.
  • Select the Make invisible menu item. The small square appears instead of the simple type data square (see Figure 2).

Figure 2. Invisible object (gray small square)

Make Visible

If you see the small square instead of the simple data type square, that means the object is made invisible. Follow the routine to make it visible:
  • Select the invisible object. The object is highlighted on the diagram (see Figure 2).
  • Right click on the selected object. The context sensitive menu appears.
  • Select the Make visible menu item. The particular simple data type square (for example STRING) appears on the diagram (see Figure 1).
