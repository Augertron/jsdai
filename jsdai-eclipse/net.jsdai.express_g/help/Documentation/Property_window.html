Property Window


There you can view properties of the selected object in the EXPRESS schema:
  • Entity
  • Definition type
  • Simple type
  • Enumeration type
  • Select type
  • Relation
  • Inter-schema Reference

Entity Properties

  • Double click or right click and select the Properties menu item on the Entity square () in the Editor panel or on the entity name next to the   icon in the Express-G Outline panel. The properties dialog window appears (see Figure 2).
  • Name field  indicates a name of the selected entity.
  • Abstract check box - indicates whether the entity is abstract super type or not..
  • Click OK or  Cancel button to close the window.

Figure 2. Entity properties dialog window

Definition Type Properties

  • Double click or right click and select the Properties menu item on the Definition type square () in the Editor panel or on the entity name next to the   icon in the Express-G Outline panel. The properties dialog window appears (see Figure 3).
  • Name field indicates a name of the selected defined type.
  • Click OK or Cancel button to close the window.

Figure 3. Definition type properties dialog window

Simple Type Properties

  • Double click or right click and select the Properties menu item on the Simple type square (). The properties dialog window appears (see Figure 4).
  • The type layout presents a list of available simple type values. The radio button in front of the type indicates the current type.
  • Click OK or Cancel button to close the window.

Relation Properties

  • Double click or right click and select the Properties menu item on the relation line ( ) in the Editor panel or on the entity name next to the   icon in the Express-G Outline panel. The properties dialog window appears (see Figure 7).
  • Name field indicates a name of the selected relationship. You can change it by entering a new value.
  • Derive, Optional, Inverse check boxes indicates a type of the relationship.

Figure 7. Relation properties dialog window
  • Also you can view the cardinality of the relationship:
    • Click on Cardinality button. The dialog window appears (see Figure 8).  
    • The Name field specifies the entity the relation shows to.
    • The radio buttons in the Type layout indicate a type of the data set to the relationship.
  • Click OK or Cancel button to close the window.

Figure 8. Relation properties dialog window (next)

Inter-schema Referenced Object Properties

  • Double click or right click and select the Properties menu item on the inter-schema reference box () in the Editor panel or on the entity name next to the   icon in the Express-G Outline panel. The properties dialog window appears (see Figure 9).
  • Name field indicates the referenced object name.
  • Alias field indicates the alias of the referenced object if any.
  • The radio list in the usage type layout indicates the usage type of the referenced object.
  • Click OK or Cancel button to close the window.
